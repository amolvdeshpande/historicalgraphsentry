This directory contains the latex guide and template for articles to the Encyclopedia of Big Data Technologies: 

1. author.tex   : the template file. If latex is used from command line, compile with sequence latex, bibtex, latex, latex.

2. author.pdf   : a compiled version of author.tex

3. author.bib   : bibtex examples used by the template 

4. svmult.cls   : the latex class file
5. spbasic.bst  : bibliography style file used by the template. Only needed if bibtex is used.

6. figure.eps   : Sample figure used
7. refguide.pdf : Reference guide for the SVMult Document Class
